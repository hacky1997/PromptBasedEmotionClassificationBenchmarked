# Sample Results

This directory contains **reference output files** from our experiment run on Google Colab (NVIDIA T4). These are provided so that:

1. Readers can verify reported numbers match the code output without running the full pipeline.
2. Downstream users can use these as templates for the expected JSON schema.

## Files

| File | Contents |
|---|---|
| `mcnemar_tests.json` | McNemar's test results (prompt vs. no-prompt) for all three models |
| `multiseed_aggregated.json` | Mean ± std accuracy and macro-F1 across seeds {42, 2024, 7} |
| `fewshot_results.json` | DistilRoBERTa accuracy at k ∈ {16, 64, 256} shots/class |
| `weighted_loss_results.json` | Standard vs. class-weighted loss comparison (ELECTRA + DistilRoBERTa) |
| `oversample_results.json` | Standard vs. oversampling comparison |
| `prompt_variant_results.json` | Template A / B / C McNemar pairwise tests |
| `classical_metrics.json` | TF-IDF + LR and LinearSVM accuracy, F1, throughput |

## Full Results

The complete set of result files (including per-model full JSON with per-sample prediction vectors, confusion matrices as `.npy`, and per-class classification reports) is generated by the notebook and saved to `MyDrive/emotion_experiment/results/`. These files are **not included here** because:

- Full JSON files are 33–35 KB each with per-sample correct vectors (2,000 booleans).
- The complete results directory contains ~25 files totalling ~800 KB.
- They are trivially reproducible by running the notebook.
